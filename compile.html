Compiling

clTEM has been compiled and tested on windows using Mingw-w64 and on linux using g++. CMake is used to create the build files. The dependencies are listed below

  • OpenCL - parallel programming language
  • clFFT - OpenCL accelerated FFTs
  • Qt5 - GUI
  • QCustomPlot - plotting
  • Boost - additional C++ libraries
  • LibTIFF - TIFF input/output
  • JSON for Modern C++ - JSON file creation/parsing
